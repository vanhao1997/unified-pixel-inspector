# Privacy Policy - Unified Pixel Inspector

**Last Updated: February 9, 2026**

## Overview

Unified Pixel Inspector is a Chrome extension designed to help marketers and developers detect, debug, and monitor tracking pixels on websites. This privacy policy explains how the extension handles data.

## Data Collection

**We do NOT collect, store, or transmit any of your data to external servers.**

All data processed by this extension:
- Stays entirely on your local device
- Is stored only in your browser's local storage
- Is never sent to any third-party servers
- Is automatically cleared when you close the browser tab

## What the Extension Accesses

The extension accesses:

1. **Page Content**: To detect installed tracking pixels (Meta, TikTok, Google, Zalo)
2. **Network Requests**: To monitor pixel firing events
3. **Local Storage**: To save your preferences (like theme settings)

## Data Security

- All sensitive parameters (email, phone, name) are automatically masked before display
- No data leaves your browser
- No external API calls are made
- No analytics or tracking of extension usage

## Permissions Explanation

| Permission | Why We Need It |
|------------|----------------|
| `activeTab` | To scan the current page for pixels |
| `scripting` | To inject detection scripts |
| `storage` | To save your settings locally |
| `webRequest` | To monitor network requests for pixel events |

## Third-Party Services

This extension does **NOT** use:
- ❌ Analytics services
- ❌ Advertising networks
- ❌ User tracking
- ❌ Data collection APIs

For questions about this privacy policy, please contact us through the Chrome Web Store listing or create an issue on our GitHub repository.
